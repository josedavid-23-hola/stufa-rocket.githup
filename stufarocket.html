¿Qué es una estufa Rocket?

Una estufa Rocket es un sistema de calefacción de alta eficiencia que utiliza combustión de madera a alta temperatura para generar calor rápidamente, optimizando el uso de combustible y reduciendo las emisiones.

Ventajas de la Estufa Rocket

  • Alta eficiencia energética
  • Reducción de emisiones contaminantes
  • Bajo consumo de madera
  • Fácil construcción y mantenimiento

Cómo construir una Estufa Rocket

Puedes construir una estufa Rocket utilizando materiales como ladrillos refractarios, tubos de acero y una base robusta. El proceso involucra crear un sistema de combustión eficiente con una cámara de combustión y un conducto de salida de humo adecuado.
